Loading and Saving Tournaments

You can load and save tournaments. They get saved as spreadsheets, in 'tab separated value' (".tsv") form. When you load a tournament by uploading such a saved file, the entrants in the tournament must already be in the database. The easiest way to make sure of this is to download the list of entrants for the tourney as well, and then upload the entrants list first. Uploading an entrant more than once is not a problem; if you upload a name that is already known the new addition is just ignored.

Editing Tips For The Tables

Click on a row to edit it. Once a row is being edited, you can save the new values with the 'Enter' key or cancel the edit with the 'Esc(ape)' key. The escape key won't work in multi-line fields like 'Notes', so to edit a note you must type in that field, then select a different field and use the 'Enter' key to save your edits.
